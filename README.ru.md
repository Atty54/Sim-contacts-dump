# Sim-contacts-dump

🇬🇧 English version: [README.md](README.md)

---

**Sim-contacts-dump** — скрипт для снятия дампов контактов с неактивных SIM-карт через 3G/4G модем с поддержкой AT-команд.  
Сценарий использования: например, при очистке старых SIM-карт дома, чтобы сохранить контакты перед утилизацией.

## Требования
- Windows 10+
- PowerShell с правами администратора
- 3G/4G модем, поддерживающий AT-команды

## Возможности
- Извлечение всех контактов с SIM
- Поддержка UCS2-кодировки для корректного отображения имён
- Сохранение дампов в текстовые файлы
- Работа с несколькими SIM-картами последовательно

## Инструкция
1. Вставьте SIM-карту в модем.  
2. Подключите модем к USB-порту.  
   - Если модем с Web-интерфейсом — переведите его в AT-режим (например, Huawei E3276, Мегафон M150-1, М100-2, МТС 822F/822FT → Project Mode).  
3. Откройте PowerShell с правами администратора.  
4. Разрешите выполнение скриптов в текущей сессии (чтобы Windows не блокировала скрипт):  
   ```powershell
   Set-ExecutionPolicy -ExecutionPolicy Process -Force
5. Выполните `devmgmt.msc`, в "Диспетчере устройств" раскройте "Порты" и запомните номер COM-порта модема.  
6. Откройте скрипт `sim_dump.ps1` в текстовом редакторе и замените `$portName = "COM6"` на ваш COM-порт. (Опционально можно изменить путь для экспорта в `$exportDir = "C:\SIM_Archive\"`).  
7. Запустите скрипт из PowerShell: `.\sim_dump.ps1`.  
8. Дождитесь окончания работы.  
9. Извлеките модем и SIM-карту.  
10. Повторяйте процесс для других SIM (шаги 5 и 6 повторять не нужно).
